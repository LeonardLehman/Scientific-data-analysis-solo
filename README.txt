The main file to generate graphs is main_runtime.ipynb python notebook. Run the notebook from start to finish to generate the used graphs except all individual country graphs.

main_runtime.ipynb dependencies:
* consumption-co2-per-capita.csv
* country_code_names.csv
* renewable-share-energy.csv
* world_bank_GDP_per_country.csv

Folder related papers contains scientific papers with similar research

Data analytics presentation end work.pdf is the presentation PDF

Methodolgy of the carbon emission numbers.pdf is the data methodology for consumption-co2-per-capita.csv data.
